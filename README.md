## ✧ Featured Projects

### ATM ClinVar + gnomAD VUS Explorer

An educational Streamlit application that combines ClinVar clinical annotations with gnomAD population-frequency evidence to explore **ATM missense variants of uncertain significance**.

**Result:** Processed 4,534 eligible ATM missense VUS candidates and matched 1,474 variants directly in gnomAD using exact GRCh38 genomic identities, producing a searchable and filterable evidence dataset.

<a href="https://github.com/luciedev166/clinvar-gnomad-vus-analyzer"><b>View project</b></a>

### E. coli oriC Finder

A Python bioinformatics project that uses cumulative GC skew and approximate DnaA-box matching to identify the likely `oriC` neighborhood in the *E. coli* K-12 MG1655 genome.

**Result:** Located a candidate region near `3.925 Mb` and detected six DnaA-box-like matches.

<a href="https://github.com/luciedev166/ecoli-ori-finder"><b>View project</b></a>

### B. subtilis oriC Finder

A follow-up Python bioinformatics project that tests whether the same cumulative GC-skew and approximate DnaA-box workflow can identify a candidate `oriC` neighborhood in the *Bacillus subtilis* strain 168 genome.

**Upgrade:** Adds circular candidate-region extraction and applies the workflow to a phylogenetically different bacterial genome.

<a href="https://github.com/luciedev166/bsubtilis-ori-finder"><b>View project</b></a>
