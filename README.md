## ✧ Featured Projects

<div align="center">

### ATM ClinVar + gnomAD VUS Explorer

An educational Streamlit application that combines ClinVar clinical annotations with gnomAD population-frequency evidence to explore **ATM missense variants of uncertain significance**.

The project includes an end-to-end pipeline for filtering ClinVar records, resolving exact GRCh38 genomic identities, integrating gnomAD population evidence, validating biological values, and assembling an app-ready dataset.

**Result:** Processed 4,534 eligible ATM missense VUS candidates and matched 1,474 variants directly in gnomAD using exact `CHROM-POS-REF-ALT` identifiers.

### E. coli oriC Finder

A Python bioinformatics project that uses cumulative GC skew and approximate DnaA-box matching to identify the likely `oriC` neighborhood in the *E. coli* K-12 MG1655 genome.

**Result:** Located a candidate region near `3.925 Mb` and detected six DnaA-box-like matches.

### B. subtilis oriC Finder

A follow-up Python bioinformatics project that applies the same cumulative GC-skew and approximate DnaA-box workflow to the *Bacillus subtilis* strain 168 genome.

**Upgrade:** Adds circular candidate-region extraction and tests the workflow on a phylogenetically different bacterial genome.
